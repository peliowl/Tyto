## 三、组件增强与修复

### 3.1 TButton

- 新增 `set_button_type()` 方法，支持运行时动态切换按钮类型
- 加载态（loading）圆形按钮优化：隐藏文字和图标，Spinner 居中显示
- Spinner 颜色跟随主题：有色背景按钮（primary/info/success/warning/error）使用白色，其他类型使用主题 primary 色
- Spinner SVG 模板化，支持动态颜色注入（`_SPINNER_SVG_TEMPLATE`）
- 圆形模式下布局边距归零、Spinner 尺寸按比例缩放
- 所有 `QColor()` 构造替换为 `parse_color()`，修复 rgba 格式解析问题

### 3.2 TTag

- 新增 `set_tag_type()` 方法，支持运行时切换颜色类型
- 新增 `set_size()` 方法，支持运行时切换尺寸变体
- 新增 `set_checkable()` 方法，支持运行时切换可勾选模式
- `set_round()` 方法改用 `apply_theme()` 替代 `_repolish()`，确保圆角正确刷新
- `sizeHint()` 改为基于 Design Token 动态计算尺寸
- `apply_theme()` 末尾增加 `updateGeometry()` 调用，确保主题切换后布局更新

### 3.3 TCheckbox

- `QColor()` 构造替换为 `parse_color()`，修复暗色模式下 rgba 颜色解析异常

### 3.4 TRadio

- `QColor()` 构造替换为 `parse_color()`，修复暗色模式下 rgba 颜色解析异常

### 3.5 TSwitch

- `QColor()` 构造替换为 `parse_color()`，修复暗色模式下 rgba 颜色解析异常

## 四、样式系统更新

### 4.1 QSS 模板新增

新增 12 个 Jinja2 QSS 模板文件（`styles/templates/`）：

`alert.qss.j2`、`backtop.qss.j2`、`card.qss.j2`、`collapse.qss.j2`、`empty.qss.j2`、`inputnumber.qss.j2`、`layout.qss.j2`、`menu.qss.j2`、`popconfirm.qss.j2`、`slider.qss.j2`、`spin.qss.j2`、`timeline.qss.j2`

### 4.2 QSS 模板修改

- **checkbox.qss.j2**：新增 `QLabel` 文字颜色继承规则，修复暗色模式下标签文字不可见问题
- **radio.qss.j2**：新增 `QLabel` 文字颜色继承规则，修复暗色模式下标签文字不可见问题
- **radiobutton.qss.j2**：新增 `QLabel` 颜色规则覆盖 checked / hover / disabled 各状态
- **tag.qss.j2**：
  - 新增 `QLabel` 背景透明与颜色继承规则
  - 各尺寸变体增加 `min-height` / `max-height` 约束
  - 圆角计算改为 `(height / 2)|int`，修复过大圆角问题
  - 新增 checkable + tagType 组合状态的完整样式覆盖（checked/unchecked × 5 种类型）
  - 新增 `strong` 模式下 `QLabel` 字重继承

### 4.3 Design Token 扩展

**light.json / dark.json** 新增以下 Token 类别：

- **颜色 Token**：`primary_suppl`、`success_suppl`、`warning_suppl`、`error_suppl`、`info_suppl`、`body_color`、`input_color`、`input_color_disabled`、`text_title`、`text_placeholder`、`divider`、`rail_color`、`close_icon`、`close_hover_bg`、`close_pressed_bg`、`alert_*`（8 个）、`timeline_*`（6 个）、`spin_mask`、`popover_color`、`indicator_color`、`hover_color`、`pressed_color`、`collapse_toggle_bg`、`collapse_toggle_border`
- **spin_sizes**：small / medium / large 旋转指示器尺寸
- **slider**：track_height、thumb_size、thumb_border_size
- **layout**：header_height、footer_height、sider_width、sider_collapsed_width、breakpoints（sm/md/lg/xl）
- **card**：padding_small / padding_medium / padding_large
- **menu**：indent、item_height、item_padding_h、collapsed_width

### 4.4 DesignTokenSet 数据模型扩展

`core/tokens.py` 新增字段：`spin_sizes`、`slider`、`layout`、`card`、`menu`，并同步更新 `to_dict()` 序列化与 `_parse_token_set()` 解析逻辑。

### 4.5 ThemeEngine 模板上下文扩展

`core/theme_engine.py` 渲染上下文新增：`spin_sizes`、`slider`、`layout`、`card`、`menu`，确保新组件 QSS 模板可引用对应 Token。
