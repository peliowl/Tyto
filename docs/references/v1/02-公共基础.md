# 公共基础

本文档描述 Tyto UI 的公共基类 `BaseWidget` 和四个行为混入（Trait Mixin）。

---

## 1. BaseWidget — 组件基类

所有 Tyto UI 组件的抽象基类，继承自 `QWidget`。自动连接 `ThemeEngine.theme_changed` 信号，确保主题切换时所有子类组件同步更新样式。

### 1.1 继承关系

```
QWidget
  └── BaseWidget
        ├── TButton
        ├── TCheckbox
        ├── TCheckboxGroup
        ├── TRadio / TRadioButton / TRadioGroup
        ├── TInput
        ├── TInputNumber
        ├── TSwitch
        ├── TTag
        ├── TSlider
        ├── TSpin
        ├── TBackTop
        ├── TEmpty
        ├── TSearchBar
        ├── TBreadcrumb
        ├── TInputGroup
        ├── TAlert
        ├── TCollapse / TCollapseItem
        ├── TPopconfirm
        ├── TTimeline / TTimelineItem
        ├── TMessage
        ├── TModal
        ├── TCard
        ├── TLayout / TLayoutHeader / TLayoutFooter / TLayoutSider / TLayoutContent
        └── TMenu / TMenuItem / TMenuItemGroup
```

### 1.2 构造函数

```python
def __init__(self, parent: QWidget | None = None) -> None
```

- 调用 `super().__init__(parent)`
- 连接 `ThemeEngine.theme_changed` → `_on_theme_changed`
- 若当前已有活跃主题，立即调用 `apply_theme()`

### 1.3 核心方法

| 方法 | 说明 |
|------|------|
| `apply_theme()` | 子类重写此方法以应用组件特定样式。默认为空操作 |
| `cleanup()` | 断开信号连接并释放资源，在组件销毁前调用 |

### 1.4 主题响应机制

```
ThemeEngine.switch_theme("dark")
       ↓ 发射 theme_changed("dark")
BaseWidget._on_theme_changed("dark")
       ↓ 委托
子类.apply_theme()
       ↓ 典型实现
engine.render_qss("button.qss.j2") → self.setStyleSheet(qss)
```

### 1.5 使用示例

```python
from tyto_ui_lib import BaseWidget, ThemeEngine

class MyWidget(BaseWidget):
    def apply_theme(self) -> None:
        engine = ThemeEngine.instance()
        bg = engine.get_token("colors", "bg_default")
        self.setStyleSheet(f"background: {bg};")
```

## 2. 行为混入（Trait Mixins）

Tyto UI 通过 Mixin 模式注入交互行为，避免在各组件中重复实现相同的视觉反馈逻辑。所有 Mixin 均设计为与 `QWidget` 子类进行协作式多重继承（Cooperative MRO）。

### 2.1 HoverEffectMixin — 悬停效果

提供 200ms 背景色过渡动画和 PointingHand 光标切换。

| 特性 | 说明 |
|------|------|
| 初始化方法 | `_init_hover_effect()` |
| 进入效果 | 光标变为 PointingHand，`QGraphicsColorizeEffect` 强度从 0 → 0.15，颜色取自 `colors.primary_hover` |
| 离开效果 | 光标恢复原始形状，colorize 强度从当前值 → 0 |
| 动画时长 | 200ms，`InOutQuad` 缓动 |
| 重写事件 | `enterEvent()`, `leaveEvent()` |

### 2.2 ClickRippleMixin — 点击涟漪

提供按下时 0.98 缩放 + 背景变暗的按压反馈效果。

| 特性 | 说明 |
|------|------|
| 初始化方法 | `_init_click_ripple()` |
| 按下效果 | 设置 `QTransform` 缩放 0.98（以控件中心为原点），设置动态属性 `pressed=True` 触发 QSS 规则 |
| 释放效果 | 恢复原始变换，设置 `pressed=False` |
| 重写事件 | `mousePressEvent()`, `mouseReleaseEvent()` |

### 2.3 FocusGlowMixin — 聚焦发光

提供 2px 半透明主色调聚焦光环效果。

| 特性 | 说明 |
|------|------|
| 初始化方法 | `_init_focus_glow()` |
| 聚焦效果 | 设置动态属性 `focused=True`，配合 QSS `[focused="true"]` 选择器渲染发光边框 |
| 失焦效果 | 设置 `focused=False`，移除发光效果 |
| 重写事件 | `focusInEvent()`, `focusOutEvent()` |

### 2.4 DisabledMixin — 禁用状态

提供 0.5 透明度 + ForbiddenCursor 的统一禁用视觉处理。

| 特性 | 说明 |
|------|------|
| 初始化方法 | `_init_disabled()` |
| 禁用效果 | `QGraphicsOpacityEffect` 透明度设为 0.5，光标变为 ForbiddenCursor，`setEnabled(False)` |
| 启用效果 | 透明度恢复 1.0，光标恢复原始形状，`setEnabled(True)` |
| 调用方法 | `set_disabled_style(disabled: bool)` |

### 2.5 ContainerQueryMixin — 容器查询

提供类似 CSS Container Query 的响应式断点系统，基于父容器 `QResizeEvent` 事件过滤器实现，无轮询。

| 特性 | 说明 |
|------|------|
| 初始化方法 | `_init_container_query()` |
| 注册断点 | `add_breakpoint(name, min_width, max_width)` |
| 查询当前断点 | `current_breakpoint() -> str \| None` |
| 安装事件过滤器 | `_install_resize_filter()` |
| 信号 | `breakpoint_changed(str)` — 断点变化时发射 |

#### 使用示例

```python
class MyWidget(ContainerQueryMixin, BaseWidget):
    def __init__(self):
        super().__init__()
        self._init_container_query()
        self.add_breakpoint("compact", 0, 400)
        self.add_breakpoint("normal", 401, 800)
        self.breakpoint_changed.connect(self._on_bp)
```

### 2.6 Mixin 使用模式

组件通过多重继承组合所需的 Mixin，并在 `__init__` 中调用各 Mixin 的初始化方法：

```python
class TButton(
    HoverEffectMixin,
    ClickRippleMixin,
    FocusGlowMixin,
    DisabledMixin,
    BaseWidget,
):
    def __init__(self, ...):
        super().__init__(parent)
        self._init_hover_effect()
        self._init_click_ripple()
        self._init_focus_glow()
        self._init_disabled()
```
