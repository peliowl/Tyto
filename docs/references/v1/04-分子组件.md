# 分子组件（Molecules）

分子级组件由原子组件组合而成，负责局部信号转发与布局协调，不包含业务逻辑。

---

## 1. TSearchBar — 搜索栏

由 `TInput` + `TButton` 组合而成的搜索栏组件，支持实时搜索和 Enter 键提交。

### 1.1 继承链

```
BaseWidget
  └── TSearchBar
```

### 1.2 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `placeholder` | `str` | `"搜索..."` | 输入框占位文本 |
| `clearable` | `bool` | `True` | 是否显示清除按钮 |
| `parent` | `QWidget \| None` | `None` | 父组件 |

### 1.3 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `search_changed` | `str` | 每次输入变化时发射 |
| `search_submitted` | `str` | 点击搜索按钮或按 Enter 时发射 |

### 1.4 公开方法

| 方法 | 说明 |
|------|------|
| `get_text() -> str` | 获取当前搜索文本 |
| `clear()` | 清空搜索输入 |

### 1.5 内部结构

```
[TInput (flex: 1)] [TButton "搜索" (Primary)]
```

- 输入框占据剩余空间（`stretch=1`）
- 搜索按钮为 Primary 类型
- 通过 `eventFilter` 拦截输入框的 Enter/Return 键事件触发提交

### 1.6 QSS 模板

模板文件：`searchbar.qss.j2`

自动处理输入框与按钮的圆角衔接：
- 输入框：右侧圆角归零，右边框移除
- 按钮：左侧圆角归零

### 1.7 使用示例

```python
bar = TSearchBar(placeholder="Search...", clearable=True)
bar.search_submitted.connect(lambda t: print(f"Search: {t}"))
```

## 2. TBreadcrumb — 面包屑导航

水平路径导航组件，支持可点击的路径项和自定义分隔符。

### 2.1 继承链

```
BaseWidget
  └── TBreadcrumb
```

### 2.2 BreadcrumbItem 数据类

```python
@dataclass
class BreadcrumbItem:
    label: str       # 显示文本
    data: Any = None # 关联的任意数据（如路由路径）
```

### 2.3 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `items` | `list[BreadcrumbItem] \| None` | `None` | 初始路径项列表 |
| `separator` | `str` | `"/"` | 分隔符字符串 |
| `parent` | `QWidget \| None` | `None` | 父组件 |

### 2.4 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `item_clicked` | `int, object` | 非末项被点击时发射，携带 (索引, data) |

### 2.5 公开方法

| 方法 | 说明 |
|------|------|
| `set_items(items: list[BreadcrumbItem])` | 替换路径项并重建 UI |
| `get_items() -> list[BreadcrumbItem]` | 获取当前路径项的副本 |

### 2.6 交互规则

- 末项（最后一个 BreadcrumbItem）：显示为当前位置，不可点击，字重 500
- 非末项：可点击，悬停时变为主色调
- 分隔符：使用 `text_disabled` 颜色

### 2.7 QSS 选择器

| 选择器 | 说明 |
|--------|------|
| `TBreadcrumb QLabel[clickable="true"]` | 可点击项 |
| `TBreadcrumb QLabel[current="true"]` | 当前位置（末项） |
| `TBreadcrumb QLabel[separator="true"]` | 分隔符 |

### 2.8 使用示例

```python
bc = TBreadcrumb(
    items=[
        BreadcrumbItem("Home", "/"),
        BreadcrumbItem("Settings", "/settings"),
        BreadcrumbItem("Profile", "/settings/profile"),
    ],
    separator=" / ",
)
bc.item_clicked.connect(lambda i, d: print(f"Navigate to: {d}"))
```

## 3. TInputGroup — 输入组

紧凑水平排列容器，自动管理子组件的边框圆角合并。

### 3.1 继承链

```
BaseWidget
  └── TInputGroup
```

### 3.2 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `parent` | `QWidget \| None` | `None` | 父组件 |

### 3.3 公开方法

| 方法 | 说明 |
|------|------|
| `add_widget(widget: QWidget)` | 追加子组件并重新计算圆角 |
| `insert_widget(index: int, widget: QWidget)` | 在指定位置插入子组件 |
| `remove_widget(widget: QWidget)` | 移除子组件 |

### 3.4 圆角合并规则

每次增删子组件后自动重新计算，圆角半径取自 `radius.medium`：

| 位置 | groupPosition 属性 | 圆角规则 |
|------|-------------------|----------|
| 唯一子组件 | `"solo"` | 保持原始圆角 |
| 第一个 | `"first"` | 仅保留左侧圆角 |
| 中间 | `"middle"` | 所有圆角归零 |
| 最后一个 | `"last"` | 仅保留右侧圆角 |

### 3.5 使用示例

```python
group = TInputGroup()
group.add_widget(TInput(placeholder="https://"))
group.add_widget(TButton("Go", button_type=TButton.ButtonType.PRIMARY))
```

## 4. TAlert — 警告提示

支持四种类型的警告提示组件，可关闭、自定义图标和操作区域。

### 4.1 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `title` | `str` | `""` | 标题文本 |
| `description` | `str` | `""` | 描述文本 |
| `alert_type` | `AlertType` | `INFO` | 类型（SUCCESS / INFO / WARNING / ERROR） |
| `closable` | `bool` | `True` | 是否可关闭 |
| `show_icon` | `bool` | `True` | 是否显示图标 |
| `bordered` | `bool` | `True` | 是否显示边框 |

### 4.2 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `closed` | 无 | 关闭动画完成后发射 |

### 4.3 公开方法

| 方法 | 说明 |
|------|------|
| `set_alert_type(alert_type)` | 设置类型 |
| `set_title(title)` | 设置标题 |
| `set_description(description)` | 设置描述 |
| `set_closable(closable)` | 设置是否可关闭 |
| `set_icon(widget)` | 自定义图标 |
| `set_action(widget)` | 设置操作区域 |

### 4.4 使用示例

```python
alert = TAlert(title="Success", description="Operation completed", alert_type=TAlert.AlertType.SUCCESS)
```

## 5. TCollapse / TCollapseItem — 折叠面板

可折叠的内容面板，支持手风琴模式和平滑展开/折叠动画。

### 5.1 TCollapseItem 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `name` | `str` | `""` | 面板唯一标识 |
| `title` | `str` | `""` | 标题文本 |
| `expanded` | `bool` | `False` | 是否初始展开 |
| `disabled` | `bool` | `False` | 是否禁用 |

### 5.2 TCollapseItem 公开方法

| 方法 | 说明 |
|------|------|
| `set_content(widget)` | 设置面板内容 |
| `set_title(widget)` | 自定义标题组件 |
| `set_header_extra(widget)` | 设置标题栏额外内容 |
| `set_arrow(widget)` | 自定义箭头组件 |
| `set_expanded(expanded, animate)` | 设置展开/折叠状态 |
| `toggle()` | 切换展开/折叠 |

### 5.3 TCollapse 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `accordion` | `bool` | `False` | 手风琴模式（同时只展开一个） |
| `arrow_placement` | `str` | `"left"` | 箭头位置（`"left"` / `"right"`） |
| `trigger_areas` | `list[str]` | `["header", "arrow"]` | 触发区域 |

### 5.4 TCollapse 公开方法

| 方法 | 说明 |
|------|------|
| `add_item(item)` | 添加折叠面板项 |
| `remove_item(item)` | 移除折叠面板项 |
| `get_expanded_names()` | 获取当前展开项的 name 列表 |

### 5.5 使用示例

```python
collapse = TCollapse(accordion=True)
item1 = TCollapseItem(name="1", title="Section 1", expanded=True)
item1.set_content(QLabel("Content 1"))
collapse.add_item(item1)
```

## 6. TPopconfirm — 气泡确认框

气泡弹出式确认框，支持多方向弹出和自定义按钮。

### 6.1 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `title` | `str` | `"确认？"` | 确认提示文本 |
| `confirm_text` | `str` | `"确认"` | 确认按钮文本 |
| `cancel_text` | `str` | `"取消"` | 取消按钮文本 |
| `placement` | `Placement` | `TOP` | 弹出方向（TOP / BOTTOM / LEFT / RIGHT） |
| `show_icon` | `bool` | `True` | 是否显示图标 |
| `trigger_mode` | `TriggerMode` | `CLICK` | 触发模式（CLICK / HOVER） |

### 6.2 公开方法

| 方法 | 说明 |
|------|------|
| `set_trigger(trigger)` | 设置触发组件 |
| `set_icon(widget)` | 自定义图标 |
| `show_popup()` | 显示弹出框 |
| `hide_popup()` | 隐藏弹出框 |

### 6.3 使用示例

```python
pop = TPopconfirm(title="Delete this item?", placement=TPopconfirm.Placement.TOP)
pop.set_trigger(TButton("Delete", button_type=TButton.ButtonType.ERROR))
pop.on_positive_click = lambda: print("Confirmed")
```

## 7. TTimeline / TTimelineItem — 时间线

垂直/水平时间线组件，支持多种布局模式和自定义节点。

### 7.1 TTimelineItem 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `title` | `str` | `""` | 标题文本 |
| `content` | `str` | `""` | 内容文本 |
| `time` | `str` | `""` | 时间文本 |
| `status` | `ItemStatus` | `DEFAULT` | 状态（DEFAULT / SUCCESS / ERROR / WARNING / INFO） |
| `color` | `str \| None` | `None` | 自定义节点颜色 |
| `line_type` | `LineType` | `SOLID` | 连线类型（SOLID / DASHED / DOTTED） |

### 7.2 TTimelineItem 公开方法

| 方法 | 说明 |
|------|------|
| `set_dot(widget)` | 自定义节点组件 |
| `set_icon(widget)` | 设置节点图标 |
| `set_title(widget)` | 自定义标题组件 |
| `set_footer(widget)` | 设置底部内容 |
| `set_line_type(line_type)` | 设置连线类型 |

### 7.3 TTimeline 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `mode` | `TimelineMode` | `DEFAULT` | 布局模式（DEFAULT / ALTERNATE / RIGHT） |
| `horizontal` | `bool` | `False` | 是否水平布局 |
| `size` | `TimelineSize` | `MEDIUM` | 尺寸变体（SMALL / MEDIUM / LARGE） |
| `icon_size` | `int` | `14` | 节点图标尺寸 |

### 7.4 TTimeline 公开方法

| 方法 | 说明 |
|------|------|
| `add_item(item)` | 添加时间线项 |
| `get_items()` | 获取所有时间线项 |
| `set_horizontal(horizontal)` | 设置水平/垂直模式 |
| `set_size(size)` | 设置尺寸 |
| `set_icon_size(size)` | 设置节点图标尺寸 |

### 7.5 使用示例

```python
timeline = TTimeline(mode=TTimeline.TimelineMode.ALTERNATE)
timeline.add_item(TTimelineItem(title="Step 1", content="Created", status=TTimelineItem.ItemStatus.SUCCESS))
timeline.add_item(TTimelineItem(title="Step 2", content="In Progress", status=TTimelineItem.ItemStatus.INFO))
timeline.add_item(TTimelineItem(title="Step 3", content="Pending", line_type=TTimelineItem.LineType.DASHED))
```
