# 原子组件（Atoms）

原子级组件是 Tyto UI 的最小可复用单元，零业务逻辑依赖，仅封装绘图逻辑与交互状态机。

---

## 1. TButton — 按钮

多功能按钮组件，支持 9 种视觉类型、4 种尺寸、多种形状和样式变体。

### 1.1 继承链

```
BaseWidget + HoverEffectMixin + ClickRippleMixin + FocusGlowMixin + DisabledMixin
  └── TButton
```

### 1.2 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `text` | `str` | `""` | 按钮标签文本 |
| `button_type` | `ButtonType` | `DEFAULT` | 视觉类型 |
| `size` | `ButtonSize` | `MEDIUM` | 尺寸变体 |
| `loading` | `bool` | `False` | 是否为加载状态 |
| `disabled` | `bool` | `False` | 是否为禁用状态 |
| `circle` | `bool` | `False` | 正圆形按钮 |
| `round` | `bool` | `False` | 全圆角（胶囊形）按钮 |
| `ghost` | `bool` | `False` | 幽灵按钮（透明背景 + 彩色边框/文字） |
| `secondary` | `bool` | `False` | secondary 层级样式 |
| `tertiary` | `bool` | `False` | tertiary 层级样式 |
| `quaternary` | `bool` | `False` | quaternary 层级样式 |
| `strong` | `bool` | `False` | 加粗文字 |
| `block` | `bool` | `False` | 撑满父容器宽度 |
| `bordered` | `bool` | `True` | 是否显示边框 |
| `color` | `str \| None` | `None` | 自定义主色调 |
| `text_color` | `str \| None` | `None` | 自定义文字颜色 |
| `icon` | `QIcon \| None` | `None` | 自定义图标 |
| `icon_placement` | `IconPlacement` | `LEFT` | 图标位置 |
| `focusable` | `bool` | `True` | 是否可聚焦 |
| `attr_type` | `AttrType` | `BUTTON` | 原生 button type |
| `parent` | `QWidget \| None` | `None` | 父组件 |

### 1.3 ButtonType 枚举

| 值 | QSS 属性值 | 说明 |
|----|-----------|------|
| `PRIMARY` | `"primary"` | 主色填充背景，白色文字 |
| `DEFAULT` | `"default"` | 白色背景，边框描边 |
| `DASHED` | `"dashed"` | 白色背景，虚线边框 |
| `TEXT` | `"text"` | 透明背景，无边框 |
| `TERTIARY` | `"tertiary"` | 浅色背景 |
| `INFO` | `"info"` | 信息色填充背景 |
| `SUCCESS` | `"success"` | 成功色填充背景 |
| `WARNING` | `"warning"` | 警告色填充背景 |
| `ERROR` | `"error"` | 错误色填充背景 |

### 1.4 ButtonSize 枚举

| 值 | 说明 |
|----|------|
| `TINY` | 超小尺寸（高度、内边距、字号由 `component_sizes.tiny` 令牌控制） |
| `SMALL` | 小尺寸 |
| `MEDIUM` | 中等尺寸（默认） |
| `LARGE` | 大尺寸 |

### 1.5 IconPlacement / AttrType 枚举

| 枚举 | 值 | 说明 |
|------|-----|------|
| `IconPlacement` | `LEFT` / `RIGHT` | 图标相对于文字的位置 |
| `AttrType` | `BUTTON` / `SUBMIT` / `RESET` | 原生 button type 属性 |

### 1.6 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `clicked` | 无 | 鼠标释放时发射（非 Loading、非 Disabled 状态下） |

### 1.7 公开方法

| 方法 | 说明 |
|------|------|
| `set_text(text)` | 更新按钮标签文本 |
| `set_size(size)` | 设置尺寸变体 |
| `set_loading(loading)` | 进入/退出加载状态 |
| `set_disabled(disabled)` | 进入/退出禁用状态 |
| `set_circle(circle)` | 设置圆形模式 |
| `set_round(round_)` | 设置全圆角模式 |
| `set_ghost(ghost)` | 设置幽灵模式 |
| `set_strong(strong)` | 设置加粗文字 |
| `set_block(block)` | 设置块级模式 |
| `set_color(color)` | 设置自定义主色调 |
| `set_text_color(text_color)` | 设置自定义文字颜色 |
| `set_bordered(bordered)` | 设置是否显示边框 |
| `set_icon(icon, placement)` | 设置图标及位置 |

### 1.8 使用示例

```python
# 基础用法
btn = TButton("Submit", button_type=TButton.ButtonType.PRIMARY)

# 不同尺寸
btn_tiny = TButton("Tiny", size=TButton.ButtonSize.TINY)
btn_large = TButton("Large", size=TButton.ButtonSize.LARGE)

# 圆形图标按钮
btn = TButton(circle=True, icon=my_icon, size=TButton.ButtonSize.SMALL)

# 幽灵按钮
btn = TButton("Ghost", button_type=TButton.ButtonType.PRIMARY, ghost=True)

# 块级按钮
btn = TButton("Full Width", block=True)
```

## 2. TCheckbox / TCheckboxGroup — 复选框

### 2.1 TCheckbox

三态复选框组件，带平滑勾选动画，支持尺寸变体和禁用状态。

#### 继承链

```
BaseWidget + HoverEffectMixin + FocusGlowMixin
  └── TCheckbox
```

#### 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `label` | `str` | `""` | 标签文本 |
| `state` | `CheckState` | `UNCHECKED` | 初始状态 |
| `size` | `CheckboxSize` | `MEDIUM` | 尺寸变体 |
| `disabled` | `bool` | `False` | 是否禁用 |
| `value` | `Any` | `None` | 在 TCheckboxGroup 中标识该选项的值 |
| `checked_value` | `Any` | `True` | 选中时的返回值 |
| `unchecked_value` | `Any` | `False` | 未选中时的返回值 |
| `default_checked` | `bool` | `False` | 默认选中状态 |
| `focusable` | `bool` | `True` | 是否可聚焦 |
| `parent` | `QWidget \| None` | `None` | 父组件 |

#### CheckState 枚举

| 值 | 整数值 | 说明 |
|----|--------|------|
| `UNCHECKED` | 0 | 未选中 |
| `CHECKED` | 1 | 已选中 |
| `INDETERMINATE` | 2 | 不确定 |

#### CheckboxSize 枚举

| 值 | 说明 |
|----|------|
| `SMALL` | 小尺寸 |
| `MEDIUM` | 中等尺寸（默认） |
| `LARGE` | 大尺寸 |

#### 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `state_changed` | `int` | 状态变化时发射 |

#### 公开方法

| 方法 | 说明 |
|------|------|
| `get_state() -> CheckState` | 获取当前状态 |
| `set_state(state)` | 设置状态 |
| `toggle()` | 在 Unchecked ↔ Checked 之间切换 |
| `get_value() -> Any` | 根据当前状态返回 checked_value 或 unchecked_value |
| `set_size(size)` | 设置尺寸变体 |
| `set_disabled(disabled)` | 设置禁用状态 |

### 2.2 TCheckboxGroup

管理一组 TCheckbox 的选中状态，支持最少/最多可选数量限制。

#### 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `value_changed` | `list` | 选中值列表变化时发射 |

#### 公开方法

| 方法 | 说明 |
|------|------|
| `add_checkbox(checkbox)` | 添加复选框到组中 |
| `get_value() -> list[Any]` | 获取当前所有选中项的 value 列表 |
| `set_value(values)` | 设置选中项 |
| `set_disabled(disabled)` | 统一禁用/启用 |
| `set_size(size)` | 统一设置尺寸 |

### 2.3 使用示例

```python
cb = TCheckbox("Accept terms", size=TCheckbox.CheckboxSize.MEDIUM)

group = TCheckboxGroup(min_count=1, max_count=3)
group.add_checkbox(TCheckbox("Option A", value="a"))
group.add_checkbox(TCheckbox("Option B", value="b"))
```

## 3. TRadio / TRadioButton / TRadioGroup — 单选按钮

### 3.1 TRadio

单选按钮组件，带环形缩放动画，支持尺寸变体和禁用状态。

#### 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `label` | `str` | `""` | 标签文本 |
| `value` | `Any` | `None` | 关联值 |
| `checked` | `bool` | `False` | 是否初始选中 |
| `size` | `RadioSize` | `MEDIUM` | 尺寸变体（SMALL / MEDIUM / LARGE） |
| `disabled` | `bool` | `False` | 是否禁用 |
| `name` | `str` | `""` | 原生 radio name |
| `parent` | `QWidget \| None` | `None` | 父组件 |

### 3.2 TRadioButton

按钮样式的单选组件，在 RadioGroup 中呈现为按钮组布局。构造参数与 TRadio 类似（无 name 参数）。

### 3.3 TRadioGroup

管理互斥逻辑。当包含 TRadioButton 子项时自动切换为按钮组样式布局。

#### 公开方法

| 方法 | 说明 |
|------|------|
| `add_radio(radio)` | 添加 TRadio 或 TRadioButton |
| `get_selected_value() -> Any` | 获取当前选中项的 value |
| `set_disabled(disabled)` | 统一禁用/启用 |
| `set_size(size)` | 统一设置尺寸 |
| `is_button_mode() -> bool` | 是否处于按钮组模式 |

### 3.4 使用示例

```python
# 传统单选
group = TRadioGroup()
group.add_radio(TRadio("Option A", value="a"))
group.add_radio(TRadio("Option B", value="b", checked=True))

# 按钮样式单选组
group = TRadioGroup()
group.add_radio(TRadioButton("Left", value="left"))
group.add_radio(TRadioButton("Center", value="center", checked=True))
group.add_radio(TRadioButton("Right", value="right"))
```

## 4. TInput — 输入框

支持 text / textarea / password 三种模式、4 种尺寸、字数统计、验证状态和加载动画。

### 4.1 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `placeholder` | `str` | `""` | 占位提示文本 |
| `input_type` | `InputType` | `TEXT` | 输入模式（TEXT / TEXTAREA / PASSWORD） |
| `size` | `InputSize` | `MEDIUM` | 尺寸变体（TINY / SMALL / MEDIUM / LARGE） |
| `clearable` | `bool` | `False` | 是否显示清除按钮 |
| `password` | `bool` | `False` | 密码模式快捷参数 |
| `round` | `bool` | `False` | 全圆角输入框 |
| `bordered` | `bool` | `True` | 是否显示边框 |
| `maxlength` | `int \| None` | `None` | 最大输入长度 |
| `show_count` | `bool` | `False` | 是否显示字数统计 |
| `readonly` | `bool` | `False` | 是否只读 |
| `loading` | `bool` | `False` | 是否显示加载动画 |
| `status` | `InputStatus \| None` | `None` | 验证状态（SUCCESS / WARNING / ERROR） |
| `rows` | `int` | `3` | textarea 默认行数 |
| `autosize` | `dict \| None` | `None` | textarea 自适应高度 |

### 4.2 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `text_changed` | `str` | 文本变化时发射 |
| `cleared` | 无 | 清除按钮被点击时发射 |

### 4.3 使用示例

```python
inp = TInput(placeholder="Enter name", clearable=True)

textarea = TInput(
    input_type=TInput.InputType.TEXTAREA,
    rows=5, show_count=True, maxlength=200,
)

pwd = TInput(input_type=TInput.InputType.PASSWORD)
```

## 5. TSwitch — 开关

支持 3 种尺寸、加载状态、方形轨道和轨道文字。

### 5.1 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `checked` | `bool` | `False` | 初始状态 |
| `disabled` | `bool` | `False` | 是否禁用 |
| `size` | `SwitchSize` | `MEDIUM` | 尺寸变体（SMALL / MEDIUM / LARGE） |
| `loading` | `bool` | `False` | 加载动画 |
| `round` | `bool` | `True` | 圆形轨道（False 为方形） |
| `checked_value` | `Any` | `True` | 开启时返回值 |
| `unchecked_value` | `Any` | `False` | 关闭时返回值 |
| `checked_text` | `str` | `""` | 开启时轨道文字 |
| `unchecked_text` | `str` | `""` | 关闭时轨道文字 |
| `rubber_band` | `bool` | `False` | 橡皮筋回弹效果 |

### 5.2 使用示例

```python
sw = TSwitch(checked_text="ON", unchecked_text="OFF", size=TSwitch.SwitchSize.LARGE)
sw = TSwitch(round=False)  # 方形轨道
```

## 6. TTag — 标签

支持 6 种颜色类型、4 种尺寸、可关闭/可选中交互和自定义颜色。

### 6.1 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `text` | `str` | `""` | 标签文本 |
| `tag_type` | `TagType` | `DEFAULT` | 颜色类型（DEFAULT / PRIMARY / SUCCESS / WARNING / ERROR / INFO） |
| `size` | `TagSize` | `MEDIUM` | 尺寸（TINY / SMALL / MEDIUM / LARGE） |
| `closable` | `bool` | `False` | 是否显示关闭按钮 |
| `round` | `bool` | `False` | 全圆角标签 |
| `disabled` | `bool` | `False` | 是否禁用 |
| `bordered` | `bool` | `True` | 是否显示边框 |
| `color` | `dict \| None` | `None` | 自定义颜色（color / border_color / text_color） |
| `checkable` | `bool` | `False` | 可选中交互 |
| `strong` | `bool` | `False` | 加粗文字 |

### 6.2 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `closed` | 无 | 关闭按钮被点击时发射 |
| `checked_changed` | `bool` | 选中状态变化时发射（仅 checkable 模式） |

### 6.3 使用示例

```python
tag = TTag("New", tag_type=TTag.TagType.PRIMARY, closable=True)
tag = TTag("Filter", checkable=True)
tag = TTag("Tiny Round", size=TTag.TagSize.TINY, round=True)
```
