# 原子组件（Atoms）

原子级组件是 Tyto UI 的最小可复用单元，零业务逻辑依赖，仅封装绘图逻辑与交互状态机。

---

## 1. TButton — 按钮

多功能按钮组件，支持四种视觉类型、Loading 状态和 Disabled 状态。

### 1.1 继承链

```
BaseWidget + HoverEffectMixin + ClickRippleMixin + FocusGlowMixin + DisabledMixin
  └── TButton
```

### 1.2 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `text` | `str` | `""` | 按钮标签文本 |
| `button_type` | `ButtonType` | `DEFAULT` | 视觉类型 |
| `loading` | `bool` | `False` | 是否初始为加载状态 |
| `disabled` | `bool` | `False` | 是否初始为禁用状态 |
| `parent` | `QWidget \| None` | `None` | 父组件 |

### 1.3 ButtonType 枚举

| 值 | QSS 属性值 | 说明 |
|----|-----------|------|
| `PRIMARY` | `"primary"` | 主色填充背景，白色文字 |
| `DEFAULT` | `"default"` | 白色背景，边框描边 |
| `DASHED` | `"dashed"` | 白色背景，虚线边框 |
| `TEXT` | `"text"` | 透明背景，无边框 |

### 1.4 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `clicked` | 无 | 鼠标释放时发射（非 Loading、非 Disabled 状态下） |

### 1.5 公开方法

| 方法 | 说明 |
|------|------|
| `set_text(text: str)` | 更新按钮标签文本 |
| `set_loading(loading: bool)` | 进入/退出加载状态，显示旋转 SVG 动画，阻止点击 |
| `set_disabled(disabled: bool)` | 进入/退出禁用状态 |

### 1.6 只读属性

| 属性 | 类型 | 说明 |
|------|------|------|
| `button_type` | `ButtonType` | 当前按钮类型 |
| `loading` | `bool` | 是否处于加载状态 |
| `text` | `str` | 当前标签文本 |

### 1.7 Loading 动画

Loading 状态下，按钮左侧显示一个 16×16 的 SVG 旋转动画（`_SpinnerWidget`）：
- 使用 `QPropertyAnimation` 驱动 `angle` 属性从 0° → 360° 循环旋转
- 动画时长 1000ms，线性缓动，无限循环
- Loading 状态下光标变为 WaitCursor，所有鼠标事件被拦截

### 1.8 QSS 选择器

模板文件：`button.qss.j2`

| 选择器 | 说明 |
|--------|------|
| `TButton` | 默认样式（Default 类型） |
| `TButton[buttonType="primary"]` | Primary 类型 |
| `TButton[buttonType="dashed"]` | Dashed 类型 |
| `TButton[buttonType="text"]` | Text 类型 |
| `TButton[focused="true"]` | 聚焦状态 |
| `TButton:hover` | 悬停伪状态 |
| `TButton:pressed` | 按下伪状态 |
| `TButton:disabled` | 禁用伪状态 |

### 1.9 使用示例

```python
btn = TButton("Submit", button_type=TButton.ButtonType.PRIMARY)
btn.clicked.connect(lambda: print("clicked"))
btn.set_loading(True)   # 进入加载状态
btn.set_loading(False)  # 退出加载状态
```

## 2. TCheckbox — 复选框

三态复选框组件，带平滑勾选动画。

### 2.1 继承链

```
BaseWidget + HoverEffectMixin + FocusGlowMixin
  └── TCheckbox
```

### 2.2 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `label` | `str` | `""` | 标签文本 |
| `state` | `CheckState` | `UNCHECKED` | 初始状态 |
| `parent` | `QWidget \| None` | `None` | 父组件 |

### 2.3 CheckState 枚举

| 值 | 整数值 | 说明 |
|----|--------|------|
| `UNCHECKED` | 0 | 未选中 |
| `CHECKED` | 1 | 已选中 |
| `INDETERMINATE` | 2 | 不确定（仅可通过代码设置） |

### 2.4 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `state_changed` | `int` | 状态变化时发射，携带新状态的整数值 |

### 2.5 公开方法

| 方法 | 说明 |
|------|------|
| `get_state() -> CheckState` | 获取当前状态 |
| `set_state(state: CheckState)` | 设置状态（触发动画和信号） |
| `toggle()` | 在 Unchecked ↔ Checked 之间切换 |

### 2.6 勾选动画

内部 `_CheckIndicator` 组件使用 `QPropertyAnimation` 驱动 `progress` 属性（0.0 → 1.0）：
- Checked 状态：绘制两段折线构成的勾选标记，动画分两阶段（0→0.5 画第一段，0.5→1.0 画第二段）
- Indeterminate 状态：绘制水平短横线
- 动画时长 150ms，`InOutQuad` 缓动

### 2.7 使用示例

```python
cb = TCheckbox("Accept terms")
cb.state_changed.connect(lambda s: print(f"State: {s}"))
cb.set_state(TCheckbox.CheckState.INDETERMINATE)
```

## 3. TRadio / TRadioGroup — 单选按钮

### 3.1 TRadio

单选按钮组件，带环形缩放动画。

#### 继承链

```
BaseWidget + HoverEffectMixin + FocusGlowMixin
  └── TRadio
```

#### 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `label` | `str` | `""` | 标签文本 |
| `value` | `Any` | `None` | 关联的任意值 |
| `checked` | `bool` | `False` | 是否初始选中 |
| `parent` | `QWidget \| None` | `None` | 父组件 |

#### 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `toggled` | `bool` | 选中状态变化时发射 |

#### 公开方法

| 方法 | 说明 |
|------|------|
| `is_checked() -> bool` | 是否选中 |
| `set_checked(checked: bool)` | 设置选中状态 |

#### 选中动画

内部 `_RadioIndicator` 使用 `QPropertyAnimation` 驱动 `progress`（0.0 → 1.0），绘制从中心向外扩展的实心圆点，最大半径 4px，颜色取自 `colors.primary`。动画时长 150ms。

### 3.2 TRadioGroup

管理一组 `TRadio` 实例的互斥逻辑。

#### 继承链

```
BaseWidget
  └── TRadioGroup
```

#### 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `selection_changed` | `object` | 选中项变化时发射，携带新选中 Radio 的 `value` |

#### 公开方法

| 方法 | 说明 |
|------|------|
| `add_radio(radio: TRadio)` | 添加单选按钮到组中 |
| `get_selected_value() -> Any` | 获取当前选中项的 value |

### 3.3 使用示例

```python
group = TRadioGroup()
group.add_radio(TRadio("Option A", value="a"))
group.add_radio(TRadio("Option B", value="b", checked=True))
group.selection_changed.connect(lambda v: print(f"Selected: {v}"))
```

## 4. TInput — 输入框

功能丰富的文本输入组件，支持前缀/后缀图标、可清除模式和密码模式。

### 4.1 继承链

```
BaseWidget + FocusGlowMixin
  └── TInput
```

### 4.2 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `placeholder` | `str` | `""` | 占位提示文本 |
| `clearable` | `bool` | `False` | 是否显示清除按钮 |
| `password` | `bool` | `False` | 是否为密码模式 |
| `prefix_icon` | `QIcon \| None` | `None` | 前缀图标 |
| `suffix_icon` | `QIcon \| None` | `None` | 后缀图标 |
| `parent` | `QWidget \| None` | `None` | 父组件 |

### 4.3 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `text_changed` | `str` | 文本内容变化时发射 |
| `cleared` | 无 | 清除按钮被点击时发射 |

### 4.4 公开方法

| 方法 | 说明 |
|------|------|
| `get_text() -> str` | 获取当前文本 |
| `set_text(text: str)` | 设置文本内容 |
| `clear()` | 清空文本并发射 `cleared` 信号 |
| `toggle_password_visibility()` | 切换密码可见性 |

### 4.5 内部布局

```
[prefix_icon?] [QLineEdit] [clear_btn?] [toggle_btn?] [suffix_icon?]
```

- 清除按钮：Unicode ✕ 字符，仅在有文本时显示
- 密码切换按钮：● (隐藏) / ○ (显示)

### 4.6 使用示例

```python
inp = TInput(placeholder="Enter name", clearable=True)
inp.text_changed.connect(lambda t: print(f"Text: {t}"))

pwd = TInput(placeholder="Password", password=True)
pwd.toggle_password_visibility()
```

## 5. TSwitch — 开关

iOS / NaiveUI 风格的滑动开关组件，带滑块缩放动画。

### 5.1 继承链

```
BaseWidget + HoverEffectMixin
  └── TSwitch
```

### 5.2 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `checked` | `bool` | `False` | 初始开关状态 |
| `disabled` | `bool` | `False` | 是否禁用 |
| `parent` | `QWidget \| None` | `None` | 父组件 |

### 5.3 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `toggled` | `bool` | 状态切换时发射 |

### 5.4 公开方法

| 方法 | 说明 |
|------|------|
| `is_checked() -> bool` | 获取当前状态 |
| `set_checked(checked: bool)` | 设置开关状态 |
| `toggle()` | 切换到相反状态 |

### 5.5 滑动动画

内部 `_SwitchTrack`（40×20px）使用 `QParallelAnimationGroup` 同时驱动两个动画：
- `thumbX`：滑块水平位置，从 3px（关）→ 23px（开），200ms `InOutCubic`
- `thumbScale`：滑块缩放，在动画中点达到 1.15 倍后回落到 1.0

绘制细节：
- 轨道：圆角矩形，开启时填充 `colors.primary`，关闭时填充 `colors.border`
- 滑块：14px 白色圆形，带 1px 偏移的半透明阴影

### 5.6 使用示例

```python
sw = TSwitch(checked=False)
sw.toggled.connect(lambda on: print(f"Switch is {'on' if on else 'off'}"))
```

## 6. TTag — 标签

分类标签组件，支持多种颜色类型、尺寸和可关闭功能。

### 6.1 继承链

```
BaseWidget
  └── TTag
```

### 6.2 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `text` | `str` | `""` | 标签文本 |
| `tag_type` | `TagType` | `DEFAULT` | 颜色类型 |
| `size` | `TagSize` | `MEDIUM` | 尺寸 |
| `closable` | `bool` | `False` | 是否显示关闭按钮 |
| `parent` | `QWidget \| None` | `None` | 父组件 |

### 6.3 TagType 枚举

| 值 | 说明 |
|----|------|
| `DEFAULT` | 默认灰色背景 |
| `PRIMARY` | 主色背景 |
| `SUCCESS` | 成功色背景 |
| `WARNING` | 警告色背景 |
| `ERROR` | 错误色背景 |

### 6.4 TagSize 枚举

| 值 | 说明 |
|----|------|
| `SMALL` | 小尺寸，padding 更紧凑，字号 12px |
| `MEDIUM` | 中等尺寸（默认），字号 14px |
| `LARGE` | 大尺寸，padding 更宽松，字号 16px |

### 6.5 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `closed` | 无 | 关闭按钮被点击时发射 |

### 6.6 公开方法

| 方法 | 说明 |
|------|------|
| `set_text(text: str)` | 更新标签文本 |

### 6.7 使用示例

```python
tag = TTag("New", tag_type=TTag.TagType.PRIMARY, closable=True)
tag.closed.connect(lambda: print("tag closed"))
```
