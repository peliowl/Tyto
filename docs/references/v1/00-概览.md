# Tyto UI Library — 技术参考文档

## 项目简介

Tyto 是一个基于 PySide6 的现代化桌面 UI 组件库，采用原子设计方法论（Atomic Design），提供从基础控件（Atom）到复杂业务模块（Organism）的分层组件体系。视觉风格参考 NaiveUI，样式系统基于 Design Token + Jinja2 + QSS 的动态渲染架构，支持 Light / Dark 主题实时无闪烁切换。

当前版本：**v1.1.0**

## 技术栈

| 类别 | 技术 |
|------|------|
| 核心框架 | PySide6 (≥ 6.7) |
| 模板引擎 | Jinja2 (≥ 3.1) |
| 构建系统 | Setuptools |
| 包管理 | uv |
| 测试框架 | pytest, pytest-qt, pytest-xdist, hypothesis |
| 代码质量 | Ruff, Pyright |
| Python 版本 | ≥ 3.12 |

## 架构总览

```
tyto_ui_lib/
├── core/               # 底层引擎：主题引擎、Design Token 数据模型、缓动引擎、事件总线
├── common/             # 公共基础：BaseWidget 基类、行为 Mixin
│   └── traits/         # 交互行为混入（Hover / Click / Focus / Disabled / ContainerQuery）
├── components/         # UI 组件（按原子设计分层）
│   ├── atoms/          # 原子级：Button, Checkbox, Radio, Input, Switch, Tag,
│   │                   #         InputNumber, Slider, Spin, BackTop, Empty
│   ├── molecules/      # 分子级：SearchBar, Breadcrumb, InputGroup,
│   │                   #         Alert, Collapse, Popconfirm, Timeline
│   └── organisms/      # 有机体：Message, Modal, Card, Layout, Menu
├── styles/             # 样式工厂
│   ├── templates/      # Jinja2 QSS 模板 (*.qss.j2)
│   └── tokens/         # 主题 Token 文件 (light.json, dark.json)
└── utils/              # 辅助工具（颜色解析等）
```

## 组件清单

### Atoms（原子级）

| 组件 | 类名 | 说明 |
|------|------|------|
| 按钮 | `TButton` | 9 种类型（Primary / Default / Dashed / Text / Tertiary / Info / Success / Warning / Error）、4 种尺寸（Tiny / Small / Medium / Large）、圆形/圆角/幽灵/块级/图标按钮，支持 Loading 和 Disabled |
| 复选框 | `TCheckbox` / `TCheckboxGroup` | 三态（Unchecked / Checked / Indeterminate）、3 种尺寸、分组管理（min/max 限制）、禁用状态 |
| 单选按钮 | `TRadio` / `TRadioButton` / `TRadioGroup` | 3 种尺寸、按钮样式变体（TRadioButton）、RadioGroup 互斥管理、禁用状态 |
| 输入框 | `TInput` | text / textarea / password 三种模式、4 种尺寸、字数统计、验证状态、加载动画、圆角、只读 |
| 数字输入框 | `TInputNumber` | 步进按钮（左右/上下布局）、范围限制（min/max）、精度控制、前缀/后缀插槽、加载/只读/验证状态、长按连续步进 |
| 滑块 | `TSlider` | 单值/范围双滑块模式、步进/标记点、垂直/水平方向、键盘操作、Tooltip、反向模式 |
| 开关 | `TSwitch` | 3 种尺寸、加载状态旋转动画、方形/圆形轨道、轨道文字、橡皮筋回弹 |
| 标签 | `TTag` | 6 种颜色类型（含 Info）、4 种尺寸（含 Tiny）、可关闭/可选中、自定义颜色、禁用状态 |
| 加载指示器 | `TSpin` | 独立/嵌套模式、ring/dots/pulse 三种动画、3 种尺寸、自定义描述文字、延迟显示、自定义图标旋转 |
| 回到顶部 | `TBackTop` | 滚动监听（QScrollArea / 自定义目标）、可控显隐、淡入淡出动画、自定义内容插槽 |
| 空状态 | `TEmpty` | 5 种尺寸（Tiny / Small / Medium / Large / Huge）、自定义图标/描述/操作区域 |

### Molecules（分子级）

| 组件 | 类名 | 说明 |
|------|------|------|
| 搜索栏 | `TSearchBar` | TInput + TButton 组合，支持 Enter 提交 |
| 面包屑 | `TBreadcrumb` | 导航路径，可点击跳转，末项为当前位置 |
| 输入组 | `TInputGroup` | 紧凑水平排列，自动合并边框圆角 |
| 警告提示 | `TAlert` | success / info / warning / error 四种类型、可关闭（淡出动画）、自定义图标和操作区域、有/无边框 |
| 折叠面板 | `TCollapse` / `TCollapseItem` | 手风琴模式、箭头位置可配（left/right）、展开/折叠平滑动画、自定义标题/箭头/额外内容 |
| 气泡确认框 | `TPopconfirm` | 多方向弹出（top/bottom/left/right）、click/hover 触发模式、自定义确认/取消按钮属性、淡入淡出动画 |
| 时间线 | `TTimeline` / `TTimelineItem` | 垂直/水平模式、alternate 交替布局、自定义节点颜色和图标、default/success/error/warning/info 状态、solid/dashed/dotted 连线类型 |

### Organisms（有机体）

| 组件 | 类名 | 说明 |
|------|------|------|
| 全局消息 | `TMessage` / `MessageManager` | Toast 风格顶部弹出消息，四种类型，自动堆叠与消失 |
| 模态对话框 | `TModal` | 遮罩层 + 居中对话框，缩放动画，可自定义标题/内容/底部 |
| 卡片 | `TCard` | 3 种尺寸（Small / Medium / Large）、可关闭、悬停阴影效果、标题/内容/底部三区域、自定义头部额外内容 |
| 布局 | `TLayout` / `TLayoutHeader` / `TLayoutFooter` / `TLayoutSider` / `TLayoutContent` | Header / Footer / Sider / Content 四区域布局，Sider 折叠动画、响应式断点（SM / MD / LG / XL） |
| 导航菜单 | `TMenu` / `TMenuItem` / `TMenuItemGroup` | 垂直/水平模式、多级子菜单弹出、折叠切换按钮、路由感知、禁用状态、缩进层级管理 |
