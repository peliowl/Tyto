# Tyto UI Library v1.0.0 — 技术参考文档

## 项目简介

Tyto 是一个基于 PySide6 的现代化桌面 UI 组件库，采用原子设计方法论（Atomic Design），提供从基础控件（Atom）到复杂业务模块（Organism）的分层组件体系。视觉风格参考 NaiveUI，样式系统基于 Design Token + Jinja2 + QSS 的动态渲染架构，支持 Light / Dark 主题实时无闪烁切换。

## 技术栈

| 类别 | 技术 |
|------|------|
| 核心框架 | PySide6 (≥ 6.7) |
| 模板引擎 | Jinja2 (≥ 3.1) |
| 构建系统 | Setuptools |
| 包管理 | uv |
| 测试框架 | pytest, pytest-qt, pytest-xdist, hypothesis |
| 代码质量 | Ruff, Pyright |
| Python 版本 | ≥ 3.12 |

## 架构总览

```
tyto_ui_lib/
├── core/               # 底层引擎：主题引擎、Design Token 数据模型
├── common/             # 公共基础：BaseWidget 基类、行为 Mixin
│   └── traits/         # 交互行为混入（Hover / Click / Focus / Disabled）
├── components/         # UI 组件（按原子设计分层）
│   ├── atoms/          # 原子级：Button, Checkbox, Radio, Input, Switch, Tag
│   ├── molecules/      # 分子级：SearchBar, Breadcrumb, InputGroup
│   └── organisms/      # 有机体：Message, Modal
├── styles/             # 样式工厂
│   ├── templates/      # Jinja2 QSS 模板 (*.qss.j2)
│   └── tokens/         # 主题 Token 文件 (light.json, dark.json)
└── utils/              # 辅助工具
```

## 组件清单

### Atoms（原子级）

| 组件 | 类名 | 说明 |
|------|------|------|
| 按钮 | `TButton` | 四种类型（Primary / Default / Dashed / Text），支持 Loading 和 Disabled 状态 |
| 复选框 | `TCheckbox` | 三态（Unchecked / Checked / Indeterminate），带勾选动画 |
| 单选按钮 | `TRadio` / `TRadioGroup` | 环形缩放动画，RadioGroup 管理互斥逻辑 |
| 输入框 | `TInput` | 支持前缀/后缀图标、可清除、密码模式 |
| 开关 | `TSwitch` | iOS / NaiveUI 风格滑动开关，带缩放动画 |
| 标签 | `TTag` | 五种颜色类型、三种尺寸、可关闭 |

### Molecules（分子级）

| 组件 | 类名 | 说明 |
|------|------|------|
| 搜索栏 | `TSearchBar` | TInput + TButton 组合，支持 Enter 提交 |
| 面包屑 | `TBreadcrumb` | 导航路径，可点击跳转，末项为当前位置 |
| 输入组 | `TInputGroup` | 紧凑水平排列，自动合并边框圆角 |

### Organisms（有机体）

| 组件 | 类名 | 说明 |
|------|------|------|
| 全局消息 | `TMessage` / `MessageManager` | Toast 风格顶部弹出消息，四种类型，自动堆叠与消失 |
| 模态对话框 | `TModal` | 遮罩层 + 居中对话框，缩放动画，可自定义标题/内容/底部 |
