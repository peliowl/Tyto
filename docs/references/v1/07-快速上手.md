# 快速上手

本文档介绍如何安装、初始化和使用 Tyto UI 组件库。

---

## 1. 安装

### 1.1 环境要求

- Python ≥ 3.12
- uv 包管理器

### 1.2 安装依赖

```bash
uv sync
```

### 1.3 开发依赖

```bash
uv sync --extra dev
```

包含：pytest, pytest-qt, pytest-xdist, hypothesis, ruff, pyright

---

## 2. 初始化主题引擎

使用任何 Tyto UI 组件前，必须先初始化 `ThemeEngine` 并激活一个主题：

```python
import sys
from PySide6.QtWidgets import QApplication
from tyto_ui_lib import ThemeEngine

app = QApplication(sys.argv)
app.setStyle("Fusion")  # 推荐使用 Fusion 基础样式

engine = ThemeEngine.instance()
engine.load_tokens()           # 加载内置 Light/Dark 令牌
engine.switch_theme("light")   # 激活 Light 主题
```

---

## 3. 使用组件

### 3.1 导入

所有公开组件均可从顶层包直接导入：

```python
from tyto_ui_lib import (
    # Atoms
    TButton, TCheckbox, TCheckboxGroup,
    TRadio, TRadioButton, TRadioGroup,
    TInput, TInputNumber, TSwitch, TTag,
    TSlider, SliderValue, TSpin,
    TBackTop, ListenToType, TEmpty,
    # Molecules
    TSearchBar, TBreadcrumb, BreadcrumbItem, TInputGroup,
    TAlert, TCollapse, TCollapseItem,
    TPopconfirm, TTimeline, TTimelineItem,
    # Organisms
    TMessage, MessageManager, MessageSlot, MessageType, TModal,
    TCard, TLayout, TLayoutHeader, TLayoutFooter, TLayoutSider,
    TLayoutContent, Breakpoint,
    TMenu, TMenuItem, TMenuItemGroup,
    # Core
    ThemeEngine, DesignTokenSet, TokenFileError, load_tokens_from_file,
    EasingEngine, EventBus,
    # Common
    BaseWidget, HoverEffectMixin, ClickRippleMixin,
    FocusGlowMixin, DisabledMixin, ContainerQueryMixin,
)
```

### 3.2 基本用法

```python
from PySide6.QtWidgets import QVBoxLayout, QWidget
from tyto_ui_lib import TButton, TInput, TCheckbox

window = QWidget()
layout = QVBoxLayout(window)

layout.addWidget(TInput(placeholder="Enter your name"))
layout.addWidget(TCheckbox("I agree to the terms"))
layout.addWidget(TButton("Submit", button_type=TButton.ButtonType.PRIMARY))

window.show()
```

### 3.3 主题切换

```python
from tyto_ui_lib import TSwitch, ThemeEngine

switch = TSwitch(checked=False)
switch.toggled.connect(
    lambda dark: ThemeEngine.instance().switch_theme("dark" if dark else "light")
)
```

## 4. Gallery 示例

项目内置了一个完整的组件预览画廊（MVVM 架构），展示所有组件的各种状态和交互效果：

```bash
uv run python examples/gallery.py
```

Gallery 采用左侧分类导航菜单 + 右侧组件特性展示面板的布局，包含以下演示区域：

| 区域 | 展示内容 |
|------|----------|
| Button | 9 种类型、4 种尺寸、圆形/圆角/幽灵/块级/图标按钮、Loading 和 Disabled |
| Checkbox | 三种状态、3 种尺寸、禁用状态、CheckboxGroup |
| Radio | RadioGroup 互斥选择、3 种尺寸、禁用状态、RadioButton 按钮组 |
| Input | 基本输入、可清除、密码、Textarea、4 种尺寸、字数统计、验证状态、圆角 |
| InputNumber | 步进按钮、范围限制、精度控制、前缀/后缀、加载/只读状态 |
| Slider | 单值/范围模式、步进/标记、垂直/水平方向 |
| Switch | 3 种尺寸、加载状态、方形开关、轨道文字 |
| Tag | 6 种颜色类型、4 种尺寸、可关闭、可选中、自定义颜色、圆角 |
| Spin | 独立/嵌套模式、三种动画类型、描述文字 |
| BackTop | 滚动监听、淡入淡出 |
| Empty | 空状态占位、自定义图标和操作区域 |
| SearchBar | 搜索栏组合组件 |
| Breadcrumb | 导航路径、自定义分隔符 |
| InputGroup | 输入框 + 按钮紧凑组合 |
| Alert | 四种类型警告提示、可关闭 |
| Collapse | 折叠面板、手风琴模式 |
| Popconfirm | 气泡确认框、多方向弹出 |
| Timeline | 时间线、交替布局 |
| Message | 四种类型的全局消息触发 |
| Modal | 模态对话框打开/关闭 |
| Card | 卡片容器、悬停阴影 |
| Layout | 布局系统、Sider 折叠 |
| Menu | 导航菜单、垂直/水平模式、多级子菜单 |

顶部工具栏提供 Dark Mode 开关，可实时切换 Light / Dark 主题。

## 5. Playground 调试应用

Playground 是一个交互式调试应用，支持实时修改控件属性并预览效果：

```bash
uv run python examples/playground.py
```

Playground 采用三栏布局：
- 左侧：分类导航菜单（与 Gallery 一致）
- 中部：组件预览区域，展示当前控件的实时实例
- 右侧：属性面板，根据属性类型自动生成编辑控件（下拉框、复选框、输入框、数字框等）

修改右侧属性面板中的任意属性值，中部预览区域会实时更新。

## 6. 测试

```bash
# 运行全部测试
uv run pytest

# 并行运行
uv run pytest -n auto

# 运行特定组件测试
uv run pytest tests/test_atoms/test_button.py

# 类型检查
uv run pyright src/

# 代码检查
uv run ruff check src/ tests/
```
