# 有机体组件（Organisms）

有机体级组件是具有独立业务上下文的复杂 UI 模块，可对接业务逻辑与数据模型。

---

## 1. TMessage / MessageManager — 全局消息

Toast 风格的全局消息系统，从屏幕顶部弹出，自动堆叠排列，支持定时自动消失。

### 1.1 MessageType 枚举

| 值 | 图标 | 说明 |
|----|------|------|
| `INFO` | ℹ | 信息提示 |
| `SUCCESS` | ✔ | 成功提示 |
| `WARNING` | ⚠ | 警告提示 |
| `ERROR` | ✖ | 错误提示 |

### 1.2 TMessage 组件

单条消息气泡组件，通常由 `MessageManager` 管理，不建议直接创建。

#### 继承链

```
BaseWidget
  └── TMessage
```

#### 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `text` | `str` | — | 消息内容 |
| `msg_type` | `MessageType` | `INFO` | 消息类型 |
| `duration` | `int` | `3000` | 自动消失延迟（ms），0 表示不自动消失 |
| `parent` | `QWidget \| None` | `None` | 父组件 |

#### 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `closed` | 无 | 关闭动画完成后发射 |

#### 公开方法

| 方法 | 说明 |
|------|------|
| `show_message()` | 显示消息并启动滑入动画和定时器 |
| `close_message()` | 关闭消息并启动淡出动画 |

#### 只读属性

| 属性 | 类型 | 说明 |
|------|------|------|
| `msg_type` | `MessageType` | 消息类型 |
| `duration` | `int` | 自动消失延迟 |
| `text` | `str` | 消息文本 |

#### 动画效果

- 显示动画：从上方滑入（250ms，`OutCubic`）
- 关闭动画：向上滑出（200ms，`InCubic`），完成后自动 `deleteLater()`

#### 窗口属性

- `FramelessWindowHint` — 无边框
- `WindowStaysOnTopHint` — 始终置顶
- `Tool` — 工具窗口类型
- `WA_TranslucentBackground` — 透明背景
- `WA_ShowWithoutActivating` — 显示时不抢焦点

### 1.3 MessageSlot 数据类

```python
@dataclass
class MessageSlot:
    message: TMessage      # 消息组件实例
    y_offset: int = 0      # 距父窗口顶部的垂直偏移
    created_at: float       # 创建时间戳
```

### 1.4 MessageManager 单例

全局消息管理器，负责消息的创建、定位和堆叠管理。

#### 类方法

| 方法 | 签名 | 说明 |
|------|------|------|
| `info` | `(text: str, duration: int = 3000) -> TMessage` | 显示信息消息 |
| `success` | `(text: str, duration: int = 3000) -> TMessage` | 显示成功消息 |
| `warning` | `(text: str, duration: int = 3000) -> TMessage` | 显示警告消息 |
| `error` | `(text: str, duration: int = 3000) -> TMessage` | 显示错误消息 |
| `instance` | `() -> MessageManager` | 获取单例 |
| `reset` | `() -> None` | 重置单例（测试用） |
| `active_slots` | `() -> list[MessageSlot]` | 获取当前活跃消息槽位列表 |

#### 堆叠规则

- 顶部边距：20px
- 消息间距：10px
- 水平居中于父窗口（无父窗口时居中于主屏幕）
- 按创建顺序从上到下排列
- 消息关闭后自动重新计算所有消息位置

### 1.5 QSS 模板

模板文件：`message.qss.j2`

| 选择器 | 说明 |
|--------|------|
| `TMessage` | 基础样式（白色背景、边框、圆角） |
| `TMessage[messageType="info"]` | 左侧 3px 蓝色边框 |
| `TMessage[messageType="success"]` | 左侧 3px 绿色边框 |
| `TMessage[messageType="warning"]` | 左侧 3px 橙色边框 |
| `TMessage[messageType="error"]` | 左侧 3px 红色边框 |
| `TMessage QLabel#message_icon` | 图标样式 |
| `TMessage QLabel#message_text` | 文本样式 |

### 1.6 使用示例

```python
from tyto_ui_lib import MessageManager

MessageManager.info("Operation completed")
MessageManager.success("File saved successfully")
MessageManager.warning("Please check your input")
MessageManager.error("Connection failed", duration=5000)
```

## 2. TModal — 模态对话框

带遮罩层的居中模态对话框，支持缩放动画和自定义内容区域。

### 2.1 继承链

```
BaseWidget
  └── TModal
```

### 2.2 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `title` | `str` | `""` | 对话框标题 |
| `closable` | `bool` | `True` | 是否显示关闭按钮 |
| `mask_closable` | `bool` | `True` | 是否允许点击遮罩关闭 |
| `parent` | `QWidget \| None` | `None` | 父组件 |

### 2.3 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `closed` | 无 | 对话框关闭时发射 |

### 2.4 公开方法

| 方法 | 说明 |
|------|------|
| `set_content(widget: QWidget)` | 设置主体内容区域 |
| `set_footer(widget: QWidget)` | 设置底部区域（通常放置操作按钮） |
| `open()` | 显示对话框并播放缩放动画 |
| `close()` | 关闭对话框并播放缩小动画 |

### 2.5 只读属性

| 属性 | 类型 | 说明 |
|------|------|------|
| `closable` | `bool` | 是否可关闭 |
| `mask_closable` | `bool` | 是否可通过遮罩关闭 |
| `title_text` | `str` | 标题文本 |

### 2.6 内部结构

```
TModal (全屏覆盖)
  └── _MaskWidget (半透明遮罩)
        └── modal_dialog (居中对话框容器，最小 400×200)
              ├── modal_header: [modal_title] [close_btn?]
              ├── modal_body: [用户自定义内容]
              └── modal_footer: [用户自定义底部] (默认隐藏)
```

### 2.7 动画效果

- 打开动画：对话框从 80% 尺寸缩放到 100%（200ms，`OutCubic`）
- 关闭动画：对话框从 100% 缩放到 80%（150ms，`InCubic`），完成后隐藏

### 2.8 窗口属性

- `FramelessWindowHint` — 无边框
- `Dialog` — 对话框类型
- `WA_TranslucentBackground` — 透明背景
- `ApplicationModal` — 应用级模态

### 2.9 QSS 模板

模板文件：`modal.qss.j2`

| 选择器 | 说明 |
|--------|------|
| `TModal QWidget#modal_mask` | 遮罩层（`colors.mask` 背景） |
| `TModal QWidget#modal_dialog` | 对话框容器（白色背景、大圆角、边框） |
| `TModal QLabel#modal_title` | 标题（大字号、加粗） |
| `TModal QPushButton#modal_close_btn` | 关闭按钮 |
| `TModal QWidget#modal_header` | 头部（底部分割线） |
| `TModal QWidget#modal_body` | 主体区域 |
| `TModal QWidget#modal_footer` | 底部（顶部分割线） |

### 2.10 使用示例

```python
from tyto_ui_lib import TModal, TButton
from PySide6.QtWidgets import QLabel, QHBoxLayout, QWidget

modal = TModal(title="Confirm", closable=True, mask_closable=True, parent=window)
modal.set_content(QLabel("Are you sure?"))

footer = QWidget()
footer_lay = QHBoxLayout(footer)
cancel = TButton("Cancel")
ok = TButton("OK", button_type=TButton.ButtonType.PRIMARY)
cancel.clicked.connect(modal.close)
ok.clicked.connect(modal.close)
footer_lay.addStretch()
footer_lay.addWidget(cancel)
footer_lay.addWidget(ok)
modal.set_footer(footer)

modal.open()
```

## 3. TCard — 卡片

带标题、内容和底部三区域的卡片容器组件，支持悬停阴影效果。

### 3.1 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `title` | `str` | `""` | 标题文本 |
| `size` | `CardSize` | `MEDIUM` | 尺寸变体（SMALL / MEDIUM / LARGE） |
| `hoverable` | `bool` | `False` | 是否启用悬停阴影效果 |
| `bordered` | `bool` | `True` | 是否显示边框 |
| `closable` | `bool` | `False` | 是否显示关闭按钮 |

### 3.2 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `closed` | 无 | 关闭按钮被点击时发射 |

### 3.3 公开方法

| 方法 | 说明 |
|------|------|
| `set_header_extra(widget)` | 设置标题栏右侧额外内容 |
| `set_content(widget)` | 设置主体内容 |
| `set_footer(widget)` | 设置底部区域 |

### 3.4 使用示例

```python
card = TCard(title="User Profile", hoverable=True, size=TCard.CardSize.MEDIUM)
card.set_content(QLabel("Card content here"))
card.set_footer(TButton("Edit", button_type=TButton.ButtonType.PRIMARY))
```

## 4. TLayout — 布局系统

提供 Header / Footer / Sider / Content 四区域布局，Sider 支持折叠动画和响应式断点。

### 4.1 Breakpoint 枚举

| 值 | 宽度阈值 | 说明 |
|----|----------|------|
| `SM` | 640px | 小屏 |
| `MD` | 768px | 中屏 |
| `LG` | 1024px | 大屏 |
| `XL` | 1280px | 超大屏 |

### 4.2 TLayoutHeader / TLayoutFooter

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `bordered` | `bool` | `True` | 是否显示边框 |
| `height` | `int \| None` | `None` | 自定义高度（默认取 Token） |

公开方法：`set_height(height)`、`set_content(widget)`

### 4.3 TLayoutSider

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `width` | `int \| None` | `None` | 展开宽度（默认取 Token） |
| `collapsed_width` | `int \| None` | `None` | 折叠宽度（默认取 Token） |
| `collapsed` | `bool` | `False` | 是否初始折叠 |
| `bordered` | `bool` | `True` | 是否显示边框 |
| `breakpoint` | `Breakpoint \| None` | `None` | 响应式断点 |

公开方法：`set_collapsed(collapsed, animate)`、`toggle_collapsed()`、`set_width(width)`、`set_collapsed_width(width)`、`set_breakpoint(bp)`、`set_content(widget)`

### 4.4 TLayoutContent

公开方法：`set_content(widget)`

### 4.5 TLayout

公开方法：`add_header(header)`、`add_footer(footer)`、`add_sider(sider)`、`add_content(content)`

### 4.6 使用示例

```python
layout = TLayout()
layout.add_header(TLayoutHeader())
layout.add_sider(TLayoutSider(collapsed=False, breakpoint=Breakpoint.MD))
layout.add_content(TLayoutContent())
layout.add_footer(TLayoutFooter())
```

## 5. TMenu — 导航菜单

支持垂直/水平模式、多级子菜单弹出、折叠切换和路由感知。

### 5.1 MenuOption 数据类

```python
@dataclass
class MenuOption:
    key: str = ""           # 菜单项唯一标识
    label: str = ""         # 显示文本
    icon: QIcon | None = None  # 关联图标
```

`MenuOption` 作为 `item_selected` 信号的载体，携带被选中菜单项的元数据。

### 5.2 MenuMode 枚举

| 值 | 说明 |
|----|------|
| `VERTICAL` | 垂直模式 |
| `HORIZONTAL` | 水平模式 |

### 5.3 TMenuItem 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `key` | `str` | — | 菜单项唯一标识 |
| `label` | `str` | `""` | 显示文本 |
| `icon` | `QIcon \| None` | `None` | 图标 |
| `disabled` | `bool` | `False` | 是否禁用 |

### 5.4 TMenuItemGroup 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `key` | `str` | — | 分组唯一标识 |
| `label` | `str` | `""` | 分组标题 |
| `icon` | `QIcon \| None` | `None` | 图标 |
| `expanded` | `bool` | `True` | 是否初始展开 |

公开方法：`add_item(item)`、`get_items()`、`set_expanded(expanded, animate)`

### 5.5 TMenu 构造参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `mode` | `MenuMode` | `VERTICAL` | 菜单模式 |
| `collapsed` | `bool` | `False` | 是否折叠 |
| `route_awareness` | `bool` | `False` | 是否启用路由感知 |
| `disabled` | `bool` | `False` | 是否禁用 |

### 5.6 信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `item_clicked` | `str` | 菜单项被点击时发射，携带 key |

### 5.7 公开方法

| 方法 | 说明 |
|------|------|
| `add_item(item)` | 添加菜单项或分组 |
| `set_active_key(key)` | 设置当前激活项 |
| `get_active_key()` | 获取当前激活项 key |
| `set_collapsed(collapsed)` | 设置折叠状态 |
| `set_mode(mode)` | 设置菜单模式 |
| `set_route(path)` | 设置路由路径（路由感知模式） |
| `set_disabled(disabled)` | 设置禁用状态 |

### 5.8 使用示例

```python
menu = TMenu(mode=TMenu.MenuMode.VERTICAL)
menu.add_item(TMenuItem(key="home", label="Home"))
group = TMenuItemGroup(key="settings", label="Settings")
group.add_item(TMenuItem(key="profile", label="Profile"))
group.add_item(TMenuItem(key="security", label="Security"))
menu.add_item(group)
menu.item_clicked.connect(lambda key: print(f"Clicked: {key}"))
```
